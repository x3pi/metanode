# Run ví dụ


`npm install`

Chọn ví dụ cần chạy sửa config chain cũng như walet nếu cần chạy lệnh sau

`npx tsx [tên file ví dụ cần chạy]`

ví dụ:

- `npx tsx getBalance.ts`
- `npx tsx getBlock.ts`
- `npx tsx getChainId.ts`
- `npx tsx getTransactionCount.ts`
- `npx tsx sendTransaction.ts`
- `npx tsx watchBlock.ts`
- `npx tsx watchBlockNumber.ts`
